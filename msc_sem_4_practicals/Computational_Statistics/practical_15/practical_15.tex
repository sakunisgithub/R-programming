\section*{\faArrowAltCircleRight[regular] \textcolor{blue}{Question}}

A random variable $X$ follows an Exponential distribution with rate parameter $\lambda = 1$. The cost associated with the system is given by $C(X) = 50 + 20 \cdot \sqrt{X}$.

\begin{enumerate}[(i)]
\item Estimate the expected cost by Monte Carlo method.

\item Estimate the expected cost by antithetic variable method.

\item Compare the variances of both the techniques.
\end{enumerate}

\section*{\faArrowAltCircleRight[regular] \textcolor{blue}{Theory}}

Given that $X \sim \text{Exp}(1)$. We have to estimate $\theta$ where the true value is

\begin{align*}
\theta &= E[C(X)] \\[0.2em]
&= \displaystyle{50 + 20 \cdot \int \limits_{0}^{\infty} e^{-x} x^{1/2} dx} \\[0.5em]
&= 50 + 20 \cdot \Gamma\left( \dfrac{3}{2} \right) \\[0.5em]
&= 67.72454.
\end{align*}

$\bullet$ If $x_1, x_2, \ldots, x_n$ is a sample of size $n$ from $X$, Monte Carlo estimate of $\theta = E[C(X)]$ is $$\widehat{\theta} = \dfrac{1}{n} \sum \limits_{i = 1}^{n} C(x_i).$$

$\bullet$ Estimate of $\theta$ by using antithetic variable will be

$$\widehat{\theta} = \dfrac{1}{n} \sum \limits_{i = 1}^{n/2} [C(X_i) + C(X_i')]$$

where $X_i$ and $X_i'$ are negatively correlated random variables. \\[0.5em]

Here $h(u) = - \ln(u)$.
